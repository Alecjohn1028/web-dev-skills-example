WEB DEVELOPMENT SKILLS

learn HTML

HTML stands for Hyper Text Markup Language HTML is the content od a web page. examples of HTML include heading, paragraphs, images, links, lists, etc.

check out theHTML tutorial on w3school.com.

Learn CSS

css stands for cascading style sheets. CSS is the style of the web page. examples of css include font styles and color, background color, borders, dimensions, layout, etc.

check out the CSS tutorial on w3schools.com.

Learn Javascript

JavaScript is the progrmming language of the web. JavaScript helps to make
web pages interactive and dynamics. Some common Javascript tasks are:

  • add new HTML to the page,
  • change the existing content /style,
  • react to user events (mouse clicks, key presses, etc.)

Check out the Javascript tutorial on javascript.info.
